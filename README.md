# SQL-project

## Overview

This project demonstrates database design and SQL query development for a fictional plumbing service company. The database manages customers, jobs, workers, skills, materials, vendors, services, and job status information.

## Skills Demonstrated

* Database design and normalization
* Primary and foreign key relationships
* Referential integrity constraints
* SQL Server database creation
* Complex JOIN operations
* Aggregate functions (COUNT, SUM, MIN, MAX)
* GROUP BY and HAVING clauses
* LEFT JOIN and RIGHT JOIN
* Data filtering using WHERE and LIKE


## Database Entities

* Customers
* Jobs
* Workers
* Skills
* Services
* Materials
* Vendors
* States
* Job Assignments
* Material Usage

## Project Contents

* Database schema creation
* Referential integrity constraints
* Sample data population
* 14 queries

## Example Reports

* In-process jobs by customer
* Material costs for completed jobs
* Total job material costs
* Worker productivity reports
* Billing rate analysis
* Unused materials
* Customers served by workers with specific skills
* Worker skill utilization
* Customer-worker job history
